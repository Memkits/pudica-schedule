
Pudica Schedule
------

> Built in ClojureScript, Respo, Cirru

Demo: http://repo.tiye.me/Memkits/pudica-schedule/

Features:

* Enter to add task
* Clear content to remove task
* Tab to focus up and down
* Scroll to move focused task
